# AutoNerve

AutoNerve is an image-classification project that uses a CNN + ViT hybrid model (PyTorch) to classify car brands and models. It includes training and evaluation code and a Streamlit-based web GUI for easy inference and demo.
<details>
  
### Version 0.1: Single Subclass CNN Training (2/19/26)

Version 0.1 focuses on single subclass training for the sake of understanding the core concepts of model training and fine tuning. In this version, I trained the model to recognize 17 different classes of Honda cars using 10000+ image datasets that were found in Kaggle.
